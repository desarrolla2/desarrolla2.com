# [Instalar el plugin de twig para netbeans](https://desarrolla2.com/post/instalar-el-plugin-de-twig-para-netbeans)

## ¿Que es twig?

twig es el motor de plantillas de moda en la comunidad de symfony2. Puedes encontrar mucha información en los sitios de [symfony2](http://symfony.com/) y de [twig](http://twig.sensiolabs.org/). El principal problema con el que me encontré es que netbeans no ofrece soporte para twig.

## ¿Cómo instalo el plugin?

Fácil, tan solo acude al [portal de plugins para netbeans](http://plugins.netbeans.org/), y descarga el [plugin de twig](http://plugins.netbeans.org/plugin/37069/php-twig). En la página del [autor](http://blogsh.de/twig-for-netbeans/), puedes encontrar más información al respecto. Una vez descargado el plugin, abre el netbeans y pulsa "tools" > "plugins" > "downloaded" > "add plugin"  > "Seleccionas el plugin" y lo instalas.[

## Características

Las características que aporta este plugin son las siguientes:

- Crear archivos twig desde las opciones de netbeans "nuevo" > "otros" > "twig html file"
- Sintaxis resaltada de HTML en los ficheros twig, si no lo tienes netbeans no colorea ni siquiera el html, y no ofrece opciones como detección de errores o formato automático.
- Sintaxis resaltada de twig, chequeo de errores, como etiquetas no cerradas, etc.
