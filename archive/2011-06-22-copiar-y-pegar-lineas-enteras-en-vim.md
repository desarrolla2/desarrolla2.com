# [Copiar y pegar lineas enteras en VIM](https://desarrolla2.com/post/copiar-y-pegar-lineas-enteras-en-vim)

Para copiar y pegar lineas de texto enteras en vim en vim, tan solo tienes que ejecutar los siguientes comandos.

Copiar:

```
yy
```

Cortar:

```
dd
```

Pegar:

```
p
```
