# [buscar y reemplazar con vim](https://desarrolla2.com/post/buscar-y-reemplazar-con-vim)

Para buscar y reemplazar cadenas de texto en vim, tan solo tienes que ejecutar el siguiente comando.

```
1,$ s/aguja/pajar/g
```
